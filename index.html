Generando secuencias en R

R permite generar secuencias de valores con el comando seq, el comando seq genera secuencias entre un valor m√≠nimo y m√°ximo y puede generar secuencias de acuerdo a cierto incremento (con el par√°metro by=) o una longitud de valores entre m√≠nimo y m√°ximo (con el par√°metro length=). Por ejemplo:

Graficando funciones con R

Una caracter√≠stica muy potente de Res en la realizaci√≥n de gr√°ficos de funciones. Para lo anterior consideraremos que una funci√≥n puede graficarse a partir de su relaci√≥n \(y=f(t)\) o \(y=f(x)\) y los valores de \(x\) los podemos generar mediante c() o seq() y los valores de \(y\) aplicando una funci√≥n a la variable \(x\). Por ejemplo, graficaremos \(y=\cos(2 \pi t)\) de \(-2\) a \(2\).

library(basetheme)  # para utilizar basetheme, primero instalarlo

y <- cos(2*pi*t)           # Calculamos la funci√≥n.
plot(t,y, type="l", col="green")
grid(col="#72efdd")      # Le agregamos lineas punteadas con color.
